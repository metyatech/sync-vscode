# Security Policy

## Supported Versions

We support security updates for the latest major version.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please report vulnerabilities by opening a GitHub Issue or contacting the maintainers.
We take all security reports seriously and will address them as quickly as possible.
In your report, please provide:
*   A description of the vulnerability.
*   Steps to reproduce.
*   Impact of the vulnerability.
