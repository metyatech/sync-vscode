# Contributing to sync-vscode

Thank you for your interest in contributing!

## How to Contribute

1.  **Report Bugs:** Use GitHub Issues.
2.  **Suggest Features:** Use GitHub Issues.
3.  **Pull Requests:**
    *   Fork the repository.
    *   Create a branch.
    *   Make changes.
    *   Ensure all tests pass (`npm run verify`).
    *   Submit a PR.

## Development

1.  Clone the repository.
2.  Run `npm install`.
3.  Run `npm run watch` to compile TypeScript.
4.  Open in VS Code and press F5 to debug.

## Compliance

This project follows [AGENTS.md](./AGENTS.md) for automated operations.
All PRs must pass CI verification.
